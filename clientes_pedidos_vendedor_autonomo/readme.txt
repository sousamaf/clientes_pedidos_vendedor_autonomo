Projeto: Controle de Clientes e Pedidos para Vendedor Autonomo

Academico Responsavel: Hirohito Pereira Goncalves;

Descricao do Projeto:
O Sistema Web para controle de clientes e pedidos para vendedores autonomos - Natura, Avon, Herbalife visa ter um 
controle geral de todos os produtos das marcas em que o vendedor operar, ter o controle dos pedidos dos clientes, e 
estabelecer uma media de compras e vendas de cada produto.
O mesmo servira como um sistema de reserva de produto onde o usuario do tipo cliente entra assim tendo acesso a todos 
os produtos cadastrados podendo fazer reservas, com isso sempre gerando um pedido com os produtos solicitados.
Cada produto ao ser comprado sera revendido e obtera um lucro. Os clientes sempre serao pessoas fisicas. Tambem tera 
uma definicao de todas as operacoes feitas ou executadas da parte financeira.

Caracteristicas:
• Gerenciamento de Clientes;
•	Gerenciamento de Pedidos;
•	Gerenciamento do Faturamento;

Rascunho Desenhado:
Usuario Master (No caso ‘eu’, pois terei acesso a tudo);
  Usuario Vendedor (Consultor da Marca);
		Marcas (Natura, Avon, Herbalife);
			Produtos (Produtos de cada marca);
				Entrada (Compra o produto para revender (Contas a Pagar));
					Venda (Revendendo o produto comprado (Contas a Receber));
					  Usuario Cliente (Sempre sera pessoa fisica).
