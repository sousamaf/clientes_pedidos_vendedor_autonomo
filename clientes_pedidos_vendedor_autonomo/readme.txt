Controle de Clientes e Pedidos para Vendedor Autônomo

Acadêmico Responsável: Hirohito Pereira Gonçalves.

Descrição do Projeto:
O Sistema Web para controle de clientes e pedidos para vendedores autônomos - Natura, Avon, Herbalife visa ter um controle geral
de todos os produtos das marcas em que o vendedor operar, ter o controle dos pedidos dos clientes, e estabelecer uma média de
compras e vendas de cada produto.
O mesmo servirá como um sistema de reserva de produto onde o usuário do tipo cliente entra assim tendo acesso a todos os produtos
cadastrados podendo fazer reservas, com isso sempre gerando um pedido com os produtos solicitados.
Cada produto ao ser comprado será revendido e obterá um lucro. Os clientes sempre serão pessoas físicas. Também terá uma definição
de todas as operações feitas ou executadas da parte financeira.

Características:
• Gerenciamento de Clientes;
• Gerenciamento de Pedidos;
• Gerenciamento do Faturamento.

Rascunho Desenhado:
Usuário Master (No caso ‘eu’, pois terei acesso a tudo);
	Usuário Vendedor (Consultor da Marca);
		Marcas (Natura, Avon, Herbalife);
			Produtos (Produtos de cada marca);
				Entrada (Compra o produto para revender (Contas a Pagar));
					Venda (Revendendo o produto comprado (Contas a Receber));
						Usuário Cliente (Sempre será pessoa física).
